# GPHH-for-UCARP-UCPP-by-Python
This project has the GPHH algorithm for solving UCARP, UCPP, URPP, and UkRPP with Python version

If you need "GPHH for UCARP" with Java version, you can go to: **https://github.com/meiyi1986/gpucarp** for more professional details. In effect, I have referenced many things from it!

If you need Python version, you can use this project.

You should open the **further_improved_genetic_programming.zip** first. The **environment.yml** file is shown directly.

The main runing file is in **further_improved_genetic_programming/GPHH_CARP/main_gphh_arp.py**.

If you want to adjust the related parameters or hyper-parameters, you can go to **further_improved_genetic_programming/GPHH_CARP/configure_parameters.py**.

This project not only includes the GPHH algorithm for solving UCARP, but UCPP(Uncertain Chinese Postman Problem). 
And other related problems are on going.
